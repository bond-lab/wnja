  • Japanese Semantic Corpus: jsemcor initial release (2012-01-06)
    • Partially sense tagged Japanese corpus. It was created by translating the SemCor corpus, and transferring the tags from English to Japanese (Bond et al. 2012).
  • 2011-05-11: Fixed bug in indexing (examples were not indexed) and definitions (lang='ENG' instead of 'eng'). Thanks to Atsushi Kato (加藤篤志) for pointing these out.
  • Version 1.1 released (2010-10-22) *bugfix release*
    • 57,238 concepts (synsets); 93,834 words; 158,058 senses (synset-word pairs)
    • 135,692 definitions; 48,276 examples
    • Many hand corrections
  • Version 1.0 released (2010-03-05)
    • 56,741 concepts (synsets); 92,241 words; 157,398 senses (synset-word pairs)
    • 135,692 definitions; 48,276 examples (NEW in 1.0)
    • Many hand corrections
  • Version 0.92 released (2009-11-16)
    • 49,655 concepts (synsets); 87,133 words; 146,811 senses (synset-word pairs).
    • Many hand corrections
  • RDF/OWL representation created by Hideaki Tanaka (2009-09-18)
  • Version 0.91 released (2009-08-31)
    • 50,739 concepts (synsets); 88,146 words; 151,831 senses (synset-word pairs).
    • Many hand corrections
    • More indexes in the SQL file
    • Ancestor table included in the SQL file
    • Links to Standard Upper Merged Ontology (SUMO)
    • Low confidence and bad links also released (wnjpn-all.tab)
    • File name changes: version removed from filenames
  • Initial Public Release (0.90) (2009-02-28)
    • 49,190 concepts (synsets); 85,966 words; 156,684 senses (synset-word pairs)
    • Links to Open Clip Art Library (OCAL)
